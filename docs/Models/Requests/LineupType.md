# LineupType

- `-1`: N/A
- `0`: Over the air
- `1`: Cable
- `2`: Satellite
- `3`: IPTV
- `4`: Virtual



## Values

| Name     | Value    |
| -------- | -------- |
| `Minus1` | -1       |
| `Zero`   | 0        |
| `One`    | 1        |
| `Two`    | 2        |
| `Three`  | 3        |
| `Four`   | 4        |
